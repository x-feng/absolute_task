this is the github repo for absolute task

1. vagrant init ubuntu/xenial64
2. vagrant up
3. vagrant ssh
4. git clone git@github.com:x-feng/absolute_task.git 
5. install flask models

    a. cd shell_scripts/flask/
    
    b. ./install_flask.sh
    
    c. ./make_app.sh
    
    d. ./run_flask.sh
6. curl http://127.0.0.1:5000
